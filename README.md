# Blind Fetch

### One-Line Test Scripts
To re-create the results in the paper, please avail of the following commands from the top level directory of the repository:

#### AUROC Graph
To generate the AUROC graph in Section 7.1, the following command can be used: <br />
```python3 ./Scripts/auroc.py ``` <br />
Which will print to the terminal the values in the table, and draw the graph.

#### IQR Box Plots.
The data for Section 7.2 is generated by running: <br />
```python3 ./Scripts/IQR.py``` <br />
This will generate both box plots in sequence, one will need to be closed before the other is shown, and the data will be printed to terminal.

#### Resource Usage
The data for Section 7.3 is generated using: <br />
```python3 ./Scripts/memory.py ``` <br />

#### Heatmaps
The two heatmaps in Section 7.4 are generated by running: <br />
```python3 ./Scripts/heatmaps.py``` <br />
This generates both heatmaps sequentially, one must be closed before the other is shown.

#### Regression
The size-based inferrence table in Section 9.3 can be generated by running: <br />
```python3 ./Scripts/regressor.py``` <br />
This prints the table contents to the folder.

#### Timing
The timing information is provided by <br />
```python3 ./Scripts/timing.py```<br />
This also will log the Averages (mean) to console.

### Formal Verification
To reproduce the symbolic proofs: <br />
```tamarin-prover --prove ./Proofs/proof.spthy``` <br />
Expected resut: All lemmas verified. One standard subterm convergence warning for OPRF correctness.
