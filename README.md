# Blind Fetch

### One-Line Test Scripts
To re-create the results in the paper, please avail of the following commands from the top level directory of the repository:

#### AUROC Graph
To generate the AUROC graph in Section 7.1, the following command can be used: 
python3 ./Scripts/auroc.py 
Which will print to the terminal the values in the table, and draw the graph.

#### IQR Box Plots.
The data for Section 7.2 is generated by running:
python3 ./Scripts/IQR.py
This will generate both box plots in sequence, one will need to be closed before the other is shown, and the data will be printed to terminal.

#### Resource Usage
The data for Section 7.3 is generated using:
python3 ./Scripts/memory.py 

#### Heatmaps
The two heatmaps in Section 7.4 are generated by running:
python3 ./Scripts/heatmaps.py
This generates both heatmaps sequentially, one must be closed before the other is shown.

#### Regression
The size-based inferrence table in Section 9.3 can be generated by running:
python3 ./Scripts/regressor.py
This prints the table contents to the folder.
